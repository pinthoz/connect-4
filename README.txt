# Connect Four

## Description

Connect Four is a two-player strategy game where players take turns dropping colored discs into a vertical grid. The objective is to connect four of your own discs vertically, horizontally, or diagonally before your opponent does. 

## Requirements Lib 

Python 3.9.1 installed.
All of the requirements are in the default python library.

## Run 

```
python3 connectfour.py <algorithm> <depth/time>
```


* [algorithm](): MiniMax, Alpha-Beta, MCTS
* [depth/time](): The depth of the search tree or the time limit for MCTS in seconds.
